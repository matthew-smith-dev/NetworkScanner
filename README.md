# NetworkScanner

**A local network scanning tool for discovering active hosts, checking ping responses, and testing common ports.**

> Status: Planning / In Development

---

## Overview

NetworkScanner demonstrates networking fundamentals used in IT support and junior networking roles.

This project is part of my IT Operations Portfolio and is aimed at building practical skills for entry-level IT support, desktop support, networking, and junior systems roles.

---

## Planned Features

- Scan a local subnet
- Ping hosts
- Display reachable IP addresses
- Hostname lookup
- Basic port checks
- Export scan results to CSV

---

## Tech Stack

- C#
- .NET
- Networking
- ICMP Ping
- TCP sockets
- CSV export
